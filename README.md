# gallery-metadata-viewer

## 起動方法

```
npm run tauri dev -- (tauriへ渡す引数) -- (cargoへ渡す引数) -- (アプリケーションへ渡す引数)
```

例: development modeで起動する

```
npm run tauri dev -- -- --no-default-features -- (ファイル)
```
